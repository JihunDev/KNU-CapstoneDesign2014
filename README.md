# KNU-CapstoneDesign2014

9축 센서를 이용해서 피겨 스케이팅 기술을 검출해 정확한 동작을 연습할 수 있도록 도와주는 시스템



## About the Project

- 안드로이드 App을 통해서 기술 확인
- 기술 점수 표시
- 화면에 발의 움직임 3D를 통해 보여줌
- 움직이는 속도 표시
- 점수 합계 표시



## Getting Started

### Requirements

- Arduino
- 9 Axis
